# Center

A widget that centers its child within itself.

## Example

```dart liveslice=Center
Container(
  height: 120.0,
  width: 120.0,
  color: Colors.blue[50],
  child: Center(
    child: FlutterLogo(
      size: 60,
    ),
  ),
)
```
