# CupertinoNavigationBar

An iOS-styled navigation bar.

## Example

```dart liveslice=CupertinoNavigationBar
CupertinoNavigationBar(
  leading: CupertinoNavigationBarBackButton(
    previousPageTitle: 'Home',
    onPressed: () {},
  ),
  middle: const Text('CupertinoNavigationBar Sample'),
)
```
