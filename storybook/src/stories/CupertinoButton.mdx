# CupertinoButton

An iOS-style button.

## Example

```dart liveslice=CupertinoButton
Column(
  mainAxisSize: MainAxisSize.min,
  children: <Widget>[
    CupertinoButton(
      child: Text('Button'),
      onPressed: () { /** */ },
    ),
    CupertinoButton.filled(
      child: Text('Button'),
      onPressed: () { /** */ },
    ),
  ],
)
```
