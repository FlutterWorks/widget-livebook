# RichText

A paragraph of rich text.

<YouTubeVideoPlayer videoId="rykDVh-QFfw" />

## Example

```dart liveslice=RichText
RichText(
  text: TextSpan(
    text: 'Hello ',
    style: DefaultTextStyle.of(context).style,
    children: <TextSpan>[
      TextSpan(text: 'bold', style: TextStyle(fontWeight: FontWeight.bold)),
      TextSpan(text: ' world!'),
    ],
  ),
)
```
