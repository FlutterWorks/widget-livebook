# SelectableText

A run of selectable text with a single style.

<YouTubeVideoPlayer videoId="ZSU3ZXOs6hc" />

## Example

```dart liveslice=SelectableText
SelectableText(
  'Hello! How are you?',
  textAlign: TextAlign.center,
  style: TextStyle(fontWeight: FontWeight.bold),
)

```
