# CupertinoTimerPicker

A countdown timer picker in iOS style.

## Example

```dart liveslice=CupertinoTimerPicker
Container(
  height: 300,
  child: CupertinoTimerPicker(
    mode: CupertinoTimerPickerMode.hms,
    onTimerDurationChanged: (value) {

    }
  ),
)
```
