# CupertinoTabBar

An iOS-styled bottom navigation tab bar.

## Example

```dart liveslice=CupertinoTabBar
CupertinoTabBar(
  items: const <BottomNavigationBarItem>[
    BottomNavigationBarItem(
      icon: Icon(CupertinoIcons.circle),
      title: Text('Tab 1'),
    ),
    BottomNavigationBarItem(
      icon: Icon(CupertinoIcons.add),
      title: Text('Tab 2'),
    ),
  ],
  currentIndex: 1,
)
```
